Lindenmeyer Systems

Originally developed by Aristid Lindenmayer in 1968, an L-Systems is a formalism to generate self-similar structure. Although L-Systems are capable of generating famous fractals like the Sierpinski Triangle (Axiom: FXF-FF-FF, F ↦ FF, X ↦ --FXF++FXF++FXF--, angle = 60) or the Gosper Curve (Axiom: F, F ↦ R, R ↦ R+L++L-R--RR-L+, L ↦ -R+LL++L+R--R-L, angle = 60), first and foremost, they were developed to model the growth of plants.
The algorithm itself is pretty simple: The model is represented by a string, in which the axiom represents the initial state of the model. On each iteration, every character of the current string (the axiom in the first iteration), is replaced by a string of characters that is defined by the according reproduction rules. E.g. Axiom: F, F-Reproduction: F ↦ FX, X-Reproduction: X ↦ F- leads to the following series of strings: F, FX, FF-, FXFX-, FF-FF--, … .
The visualization is generated by interpreting the current string as a turtle-graphic. F means: Move forward by a constant length and draw a line; f means: Move forward by the same constant length and don't draw a line; + means: Turn right by a given angle, - means: Turn left by the same angle. The symbol [ store the current position and angle, whereas the symbol ] restores the last position and angle, i.e. jumps back to the last ]. Every other character is simply ignored. For a more sophisticated explanation of L-Systems turn to "The Algorithmic Beauty of Plants" by Przemysław Prusinkiewicz and Aristid Lindenmayer.
